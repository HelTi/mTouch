## a touch js library for mobile like jquery
> mTouchjs分离自jquery

### 使用方法和juqery一样
```javascript

 mTouch("selector").css()
 mTouch.each(obj,function(){})
 mTouch.each('selector',function(i,j) {
   console.log(mTouch(this))
 })
```
